# Lendsqr Adjutor API Testing

## Overview
This project contains API testing for Lendsqr Adjutor services.

## Contents
- Postman Collection (Automated Tests)
- Environment File
- Test Cases Documentation
- API Documentation

## How to Run
1. Import collection into Postman
2. Import environment file
3. Set access_token
4. Run using Collection Runner

## Tools Used
- Postman
- JavaScript (Postman scripts)

## Key Highlights
- API discovery using browser dev tools
- Automated test scripts
- Handling async API (empty 200 response case)
